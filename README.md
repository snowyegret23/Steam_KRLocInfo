# KOSTEAM

Steam 상점 페이지에서 한국어 패치 정보를 표시하고, 언어 필터를 우회하는 브라우저 확장 프로그램

[![Chrome](https://developer.chrome.com/static/docs/webstore/branding/image/UV4C4ybeBTsZt43U4xis.png)](https://chromewebstore.google.com/detail/cbcganplchikpgchgppklcblpjibhnnc) 
[![Firefox](https://blog.mozilla.org/addons/files/2015/11/get-the-addon.png)](https://addons.mozilla.org/ko/firefox/addon/kosteam/)  

## 주요 기능

### 한국어 패치 정보 표시

- Steam 상점 페이지에서 한국어 지원 상태를 자동으로 표시
- 공식 한국어 / 공식(추가정보 존재) / 공식지원 추정 / 다이렉트 게임즈 / 스토브 / 유저패치 / 한국어 없음 구분
- 패치 정보 바로가기 링크 제공

### 검색 언어설정 우회

- Steam 검색 시 언어 필터로 숨겨진 게임들을 모두 표시
- 카테고리, 장르, 태그 페이지에서도 자동 우회
- 상점 페이지의 리뷰 언어 필터는 현재 설정 유지

### 장바구니 선택 구매

- '선택항목만 남기기'로 결제 후 '복원'을 누르면 삭제된 항목 복원
- 장바구니 항목을 찜 목록에 추가 / JSON으로 내보내기 (백업 기능)
- 기능 오류 시 새로고침 권장 (Steam API 불안정성 이슈)

### Steam 커뮤니티에서 상점으로 이동

- Steam 커뮤니티 페이지에 '상점 페이지' 버튼이 누락된 경우, '상점으로 이동' 버튼을 표시

## 패치 정보 수집 방식

본 프로젝트는 외부 웹사이트로부터 한국어 패치 정보를 수집하며, 윤리적이고 투명한 수집 정책을 준수합니다.

**사용자의 개인정보는 일체 수집하지 않습니다.** 개인정보 보호 정책은 [PRIVACY.md](PRIVACY.md)를 참고해주세요.

### 수집 방식
- **신원 공개**: 모든 요청에 프로젝트 식별자(`KOSTEAM-Webscraper/1.0`)를 포함하여 투명하게 운영합니다.
- **연락처 제공**: 요청 헤더에 GitHub 저장소 주소를 포함하여 웹사이트 운영자가 연락 가능하도록 합니다.
- **서버 부하 최소화**: 요청 간격 제한(0.5초~2초)을 적용하여 서버에 무리를 주지 않습니다.

### 데이터 사용
- **비상업적 용도**: 무료 오픈소스 확장 프로그램에만 사용하며, 상업적 목적의 사용이나 재판매를 하지 않습니다.
- **출처 명시**: 모든 데이터의 출처(스팀앱, 퀘이사존 큐레이터 등)를 명확히 표시합니다.
- **링크 제공 방식**: 외부 사이트의 콘텐츠를 복제하지 않고, 해당 사이트로의 링크만 제공합니다.

### 중단 요청
웹사이트 운영자께서 데이터 수집 중단을 원하시면 [Issues](https://github.com/snowyegret23/KOSTEAM/issues)로 연락 주시기 바랍니다. 즉시 해당 사이트에 대한 데이터 수집을 중단하겠습니다.
