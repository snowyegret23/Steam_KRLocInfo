# 개인정보처리방침

최종 수정일: 2025년 1월 11일

## 개요

KOSTEAM(이하 "확장 프로그램")은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

## 수집하는 정보

이 확장 프로그램은 개인정보를 수집하지 않습니다.

## 로컬 저장 데이터

확장 프로그램은 Chrome의 storage API를 사용하여 다음 데이터를 사용자의 기기에만 저장합니다:

- 사용자 설정 (표시할 패치 소스 선택, 검색 언어필터 우회 설정)
- 캐시된 한국어 패치 정보 데이터

이 데이터는 외부 서버로 전송되지 않으며, 사용자의 브라우저 내에서만 사용됩니다.

## 외부 통신

확장 프로그램은 다음 목적으로만 외부와 통신합니다:

- GitHub (raw.githubusercontent.com): 한국어 패치 정보 데이터(JSON) 다운로드

이 통신에서 사용자의 개인정보는 전송되지 않습니다.

## 제3자 제공

수집하는 개인정보가 없으므로 제3자에게 제공하는 정보도 없습니다.
